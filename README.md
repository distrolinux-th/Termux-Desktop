# Termux-X11-Debian

🚀 One Command Installer for Termux:X11 + Proot-Distro Debian + XFCE

## Features

- One Command Install
- Termux:X11
- Debian (proot-distro)
- XFCE Desktop
- PulseAudio
- ZSH + Oh My Zsh
- Powerlevel10k
- Windows 10/11 Theme
- VSCodium
- Auto Start Desktop
- Auto Repair

## Installation

```bash
bash <(curl -fsSL https://raw.githubusercontent.com/linuxdistro1/Termux-X11-Debian/main/install.sh)
```
